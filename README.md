# 🛒 E-commerce Database Design (MySQL)

## 📌 Project Overview
This project is a complete relational database design for an E-commerce system built using MySQL. It simulates real-world online shopping operations including customers, products, orders, payments, and addresses.

---

## 🧱 Database Structure
The database consists of the following tables:
- Customer
- Products
- Orders
- Order_Items
- Payments
- Address

Each table is connected using **Primary Keys and Foreign Keys** to maintain data integrity.

---

## ⚙️ Features
- Normalized relational database design
- Realistic sample data
- Proper use of constraints (PK, FK, UNIQUE)
- Multi-table relationships
